# ResistanceMap Dashboard

A React static dashboard for the ResistanceMap pipeline with model diagrams, interpretability flows, and governance summaries.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## GitHub Pages

This repo is configured for GitHub Pages with the base path `/dashboard/` in `vite.config.js`.

To publish manually:

```bash
npm run build
```

Then enable GitHub Pages for the `gh-pages` branch or configure the included workflow.
